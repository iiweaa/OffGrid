# M4-T8 Beta 版本测试方案

> 验证目标：在 M4 功能（除已删除的省电模式外）全部落地后，对一加 11 / 华为 Mate 30 Pro 5G 进行多场景 Beta 测试，确认无 P0/P1 缺陷，输出 Beta 测试报告。

---

## 1. 测试环境

| 项目 | 要求 |
|------|------|
| 设备 A | 一加 11（`62978a15`），Android 14 |
| 设备 B | 华为/荣耀 Mate 30 Pro 5G（`2KE0219B02018194`），HarmonyOS / Android 12 |
| 应用版本 | 最新 `main` 分支 debug APK（`04efdb9` 及之后） |
| 网络 | 关闭移动数据与 Wi-Fi，仅使用 Wi-Fi Direct |
| 权限 | 已授予位置、附近设备、录音、通知权限 |
| 电量 | 每轮起始电量 ≥ 30% |

---

## 2. 测试用例

### BC-1 基础通话（GO + Client）

| 步骤 | 设备 A | 设备 B | 预期结果 |
|------|--------|--------|----------|
| 1 | Settings → Connection → Group Owner | Settings → Connection → Client | — |
| 2 | Call → Start Call | — | A 创建 Group，状态为 Connected |
| 3 | — | Call → Start Call | B 发现并加入 Group，状态为 Connected |
| 4 | 双向通话 2 分钟 | 双向通话 2 分钟 | 双向语音清晰，Neighbors 显示对端 NodeID/IP |
| 5 | 点击 End Call | 点击 End Call | 通话结束，状态回到 Idle |

### BC-2 角色互换（Client + GO）

| 步骤 | 设备 A | 设备 B | 预期结果 |
|------|--------|--------|----------|
| 1 | Client | Group Owner | — |
| 2 | — | 先 Start Call | B 创建 Group |
| 3 | 后 Start Call | — | A 发现并加入，通话正常 |

### BC-3 锁屏/后台通话

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 按 BC-1 建立通话 | 通话正常 |
| 2 | 双方按电源键锁屏，保持 5 分钟 | 前台服务存活，语音不中断，无断连 |
| 3 | 点亮屏幕 | 应用状态正确，可正常结束通话 |

### BC-5 长时间稳定性

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 按 BC-1 建立通话 | 通话正常 |
| 2 | 保持通话 15 分钟 | 无 ANR、无 Crash、无断连；语音持续可用 |

### BC-6 应用被杀后重连

| 步骤 | 操作 | 预期结果 |
|------|------|----------|
| 1 | 按 BC-1 建立通话 | 通话正常 |
| 2 | 在最近任务中上滑关闭 A 端应用 | B 端提示对端离线/邻居消失 |
| 3 | 重新打开 A 端，重新选择 Group Owner 并 Start Call | B 端重新发现并连接，通话恢复 |

### BC-7 权限边界

| 场景 | 操作 | 预期结果 |
|------|------|----------|
| 录音权限被拒绝 | 系统设置撤销录音权限后进入 Call 点击 Start Call | 提示权限不足，不崩溃 |
| 位置权限被拒绝 | 撤销位置权限后点击 Start Call | 提示权限不足，不崩溃 |

### BC-8 音频外设（可选）

| 场景 | 操作 | 预期结果 |
|------|------|----------|
| 有线耳机 | 通话中插入有线耳机 | 音频自动切换到耳机，通话继续 |
| 蓝牙耳机 | 通话前连接蓝牙耳机 | 音频从蓝牙耳机播放与采集，通话正常 |
| 耳机拔出 | 通话中拔出耳机 | 音频切换回扬声器，通话继续 |

> 注：若当前没有有线/蓝牙耳机，可在「实际结果」中标注 N/A。

## 3. 缺陷等级定义

| 等级 | 定义 | 示例 |
|------|------|------|
| P0 | 阻塞性问题，必须立即修复 | 应用 Crash、ANR、无法建立任何通话 |
| P1 | 严重影响体验，Beta 验收前需修复 | 锁屏后断连、角色切换失败、音频外设无法切换 |
| P2 | 一般问题，可记录到 Issue 后续优化 | UI 显示延迟、日志冗余、提示文案不友好 |
| P3 | 建议/优化 | 动画、文案微调 |
